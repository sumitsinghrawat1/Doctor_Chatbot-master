# Doctor_Chatbot-master

AI Healthcare Chatbot (Heart Disease Assistant)

## Overview
This project is an AI-based healthcare chatbot developed to assist users in monitoring heart-related health conditions. The chatbot interacts with users and provides basic health predictions using Machine Learning techniques.

## Features
- Heart disease prediction
- User-friendly chatbot interaction
- Machine Learning-based analysis
- Simple and responsive interface

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript

## Purpose of the Project
The main objective of this project is to provide an intelligent healthcare assistance system that helps users understand possible heart-related risks and encourages early awareness.

## Future Improvements
- Real-time health monitoring
- Appointment scheduling
- Improved AI responses
- Cloud deployment
